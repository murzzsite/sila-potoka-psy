# Анна — психолог интегративного подхода

Лендинг для психолога. Интегративный подход: КПТ, системные расстановки, терапия души, игра Лила.
Темы: тревожность, отношения, проявленность, деньги.

- Координаты карты — заглушка (центр Москвы), заменить на актуальные, если нужна карта
- Отзывы — примеры, заменить на реальные при появлении
